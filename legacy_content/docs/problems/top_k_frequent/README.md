# top_k_frequent

## Overview

Return the k most frequent elements in an integer list.

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
top k frequent: 4 1
```

## Run with project runner

From repository root:

```powershell
& ".\tools\run_example.exe" -p top_k_frequent -l cpp20
& ".\tools\run_example.exe" -p top_k_frequent -l go
& ".\tools\run_example.exe" -p top_k_frequent -l python
& ".\tools\run_example.exe" -p top_k_frequent -l scala
& ".\tools\run_example.exe" -p top_k_frequent -l typescript
```
