# 17.12. BiNode

## Description

Given a binary search tree, convert it to a sorted linked list in-place.
Use the right pointer as the next pointer and set all left pointers to `null`.

**Example:**

```
Input:  BST nodes [4,2,5,1,3,null,6,0]
Output: 0 -> 1 -> 2 -> 3 -> 4 -> 5 -> 6
```
