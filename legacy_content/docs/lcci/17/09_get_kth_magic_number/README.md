# 17.09. Get Kth Magic Number

## Description

A magic number is a positive integer whose prime factors are only `3`, `5`, and `7`.
Design an algorithm to find the `k`th magic number.

**Example 1:**

```
Input: k = 5
Output: 9
```

**Example 2:**

```
Input: k = 1
Output: 1
```
