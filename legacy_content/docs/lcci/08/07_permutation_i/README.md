# 08.07 Permutation I

## Description

Write a method to compute all permutations of a string of unique characters.

**Example:**
```
Input: S = "qwe"
Output: ["qwe","qew","wqe","weq","ewq","eqw"]
```

**Note:** All characters in the string are unique. The length of the string won't exceed 8.

```

qew

```
