# 08.11 Coin

## Description

Given an infinite number of quarters (25 cents), dimes (10 cents), nickels (5 cents), and pennies (1 cent), write code to calculate the number of ways of representing n cents. (The result may be large, so return it modulo 1000000007)

**Example 1:**
```
Input: n = 5
Output: 2
Explanation: There are two ways: {5} and {1,1,1,1,1}
```

**Example 2:**
```
Input: n = 10
Output: 4
```

**Note:** `0 <= n <= 1000000`
