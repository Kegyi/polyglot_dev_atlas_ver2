# 08.12 Eight Queens

## Description

Write an algorithm to print all ways of arranging n queens on an n×n chess board so that none of them share the same row, column, or diagonal. Write a method to return the arrangement as an array of strings.

**Example:**
```
Input: n = 4
Output: [
 [".Q..","...Q","Q...","..Q."],
 ["..Q.","Q...","...Q",".Q.."]
]
```

**Note:** `1 <= n <= 9`
