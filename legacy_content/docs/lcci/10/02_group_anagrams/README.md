# 10.02 Group Anagrams

## Description

Sort an array of strings so that all anagrams are grouped together.

**Example:**
```
Input: ["eat", "tea", "tan", "ate", "nat", "bat"]
Output:
[
  ["ate", "eat", "tea"],
  ["bat"],
  ["nat", "tan"]
]
```

**Notes:**
- All inputs are lowercase.
- Output order does not matter.
