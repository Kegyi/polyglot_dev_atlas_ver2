# 10.05 Sparse Array Search

## Description

Given a sorted array of strings interspersed with empty strings, find the index of `s`.

**Example 1:**
```
Input: words = ["at", "", "", "", "ball", "", "", "car", "", "", "dad", "", ""], s = "ta"
Output: -1
```

**Example 2:**
```
Input: words = ["at", "", "", "", "ball", "", "", "car", "", "", "dad", "", ""], s = "ball"
Output: 4
```
