# 10.11 Peaks and Valleys

## Description

Reorder an array into an alternating sequence of peaks and valleys, where a peak is greater than or equal to its neighbors and a valley is less than or equal to its neighbors.

**Example:**
```
Input: [5, 3, 1, 2, 3]
One valid output: [2, 1, 3, 3, 5]
```
