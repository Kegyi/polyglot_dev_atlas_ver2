# 03.01 Three in One

## Description

Describe how to implement three stacks using one array. This maps to LCCI 3.1.

Typical interview angle:
- Reserve equal-capacity segments for each stack.
- Maintain top pointers per stack.
- Return sentinel values for underflow.

Reference page: https://leetcode.doocs.org/en/lcci/3.1/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
11
11
10
-1
false
```
