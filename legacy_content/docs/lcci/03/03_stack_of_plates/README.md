# 03.03 Stack of Plates

## Description

Design a set-of-stacks where each sub-stack has a fixed capacity and behaves like one big stack. This maps to LCCI 3.3.

Typical interview angle:
- Start a new sub-stack on overflow.
- Remove empty sub-stacks after pop/popAt.
- Discuss rollover trade-offs if strict balancing is required.

Reference page: https://leetcode.doocs.org/en/lcci/3.3/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
2
4
3
1
```
