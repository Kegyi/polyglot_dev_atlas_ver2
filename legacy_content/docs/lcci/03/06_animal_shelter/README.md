# 03.06 Animal Shelter

## Description

Model an animal shelter that dequeues the oldest animal overall or the oldest of a specific type. This maps to LCCI 3.6.

Typical interview angle:
- Keep separate queues for dogs and cats.
- Attach a global arrival timestamp.
- DequeueAny compares queue fronts.

Reference page: https://leetcode.doocs.org/en/lcci/3.6/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
1
2
3
-1
```
