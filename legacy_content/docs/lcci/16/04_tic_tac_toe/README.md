# 16.04 Tic-Tac-Toe

## Description

Given an `N x N` tic-tac-toe board containing `'X'`, `'O'`, and `' '` (empty), return:
- `"X"` or `"O"` if that player has won
- `"Pending"` if no winner and at least one empty cell exists
- `"Draw"` otherwise

- [Python](python.py)
- [Go](go.go)
