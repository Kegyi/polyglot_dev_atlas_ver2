# 16.13 Bisect Squares

## Description

Given two axis-aligned squares represented by `[left, bottom, size]`, return the segment endpoints of the line that cuts both squares into two equal halves.

## Expected output

```
[-1.0, 0.0, 2.0, 0.0]
```
