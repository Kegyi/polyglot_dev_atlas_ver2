# 01.01 Is Unique

## Description

Given a string, determine whether every character appears at most once. This maps to LCCI 1.1 from the linked source and focuses on fast duplicate detection with linear scanning.

Typical interview angle:
- Can you solve it in O(n) time?
- What changes if extra data structures are disallowed?
- How does character set size (ASCII vs Unicode) affect the implementation?

Reference page: https://leetcode.doocs.org/en/lcci/1.1/
