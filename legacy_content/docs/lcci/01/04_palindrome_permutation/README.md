# 01.04 Palindrome Permutation

## Description

Determine whether any permutation of a string can form a palindrome. The key property is that at most one character may have an odd frequency. This maps to LCCI 1.4.

Typical interview angle:
- Normalize case and handle spaces consistently.
- Reduce the problem to parity of character counts.
- Present an O(n) counting approach.

Reference page: https://leetcode.doocs.org/en/lcci/1.4/

- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
tact coa -> true
daily -> false
```
