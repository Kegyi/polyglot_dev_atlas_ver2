# 02.01 Remove Duplicate Node

## Description

Remove duplicate values from an unsorted linked list while keeping the first appearance of each value. This maps to LCCI 2.1.

Typical interview angle:
- Hash-set solution in O(n) time and O(n) extra space.
- Discuss the no-buffer variant with O(n^2) time.
- Careful pointer updates while deleting nodes.

Reference page: https://leetcode.doocs.org/en/lcci/2.1/

- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

