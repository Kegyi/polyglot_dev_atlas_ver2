# 02.07 Intersection of Two Linked Lists

## Description

Find the first common node of two singly linked lists. This maps to LCCI 2.7.

Typical interview angle:
- Two-pointer switching trick: each pointer traverses both lists.
- If intersection exists, pointers align after at most two passes.
- If not, both pointers end at null.

Reference page: https://leetcode.doocs.org/en/lcci/2.7/
