# 02.08 Linked List Cycle

## Description

Determine whether a linked list contains a cycle. This maps to LCCI 2.8.

Typical interview angle:
- Floyd's tortoise-and-hare cycle detection.
- $O(1)$ extra space.
- Works without modifying nodes.

Reference page: https://leetcode.doocs.org/en/lcci/2.8/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
true
false
```
