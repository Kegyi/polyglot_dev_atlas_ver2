# 04.08 First Common Ancestor

## Description

Find the lowest common ancestor of two nodes in a binary tree. This maps to LCCI 4.8.

Typical interview angle:
- Return current node when subtree contains both targets.
- Bubble up partial matches from left and right recursion.
- Clarify behavior when one or both nodes do not exist.

Reference page: https://leetcode.doocs.org/en/lcci/4.8/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
5
3
```
