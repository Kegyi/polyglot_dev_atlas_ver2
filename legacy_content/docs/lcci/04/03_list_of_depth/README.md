# 04.03 List of Depth

## Description

Given a binary tree, create a linked-list-like collection of nodes at each depth. This maps to LCCI 4.3.

Typical interview angle:
- Level-order traversal (BFS).
- One container per depth.
- Time complexity linear in node count.

Reference page: https://leetcode.doocs.org/en/lcci/4.3/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
1 | 2 3 | 4 5 6 7
```
