# 04.05 Legal Binary Search Tree

## Description

Determine whether a binary tree satisfies BST ordering rules. This maps to LCCI 4.5.

Typical interview angle:
- Carry valid (low, high) bounds recursively.
- Every node must satisfy strict inequality.
- Catch deeper violations, not only parent-child checks.

Reference page: https://leetcode.doocs.org/en/lcci/4.5/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
true
false
```
