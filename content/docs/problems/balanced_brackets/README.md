# balanced_brackets

## Overview

Check whether bracket sequences are balanced using a stack.

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
input_1 valid: true
input_2 valid: false
```

## Run with project runner

From repository root:

```powershell
& ".\tools\run_example.exe" -p balanced_brackets -l cpp20
& ".\tools\run_example.exe" -p balanced_brackets -l go
& ".\tools\run_example.exe" -p balanced_brackets -l python
& ".\tools\run_example.exe" -p balanced_brackets -l scala
& ".\tools\run_example.exe" -p balanced_brackets -l typescript
```
