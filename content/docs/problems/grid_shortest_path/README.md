# grid_shortest_path

## Overview

Find the shortest path in a grid from top-left to bottom-right using BFS.

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
shortest steps: 8
```

## Run with project runner

From repository root:

```powershell
& ".\tools\run_example.exe" -p grid_shortest_path -l cpp20
& ".\tools\run_example.exe" -p grid_shortest_path -l go
& ".\tools\run_example.exe" -p grid_shortest_path -l python
& ".\tools\run_example.exe" -p grid_shortest_path -l scala
& ".\tools\run_example.exe" -p grid_shortest_path -l typescript
```
