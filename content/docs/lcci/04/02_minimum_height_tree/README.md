# 04.02 Minimum Height Tree

## Description

Given a sorted array, build a binary search tree with minimal height. This maps to LCCI 4.2.

Typical interview angle:
- Pick middle element as root recursively.
- Build left and right subtrees from halves.
- Validate with inorder traversal.

Reference page: https://leetcode.doocs.org/en/lcci/4.2/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
1 2 3 4 5 6 7
```
