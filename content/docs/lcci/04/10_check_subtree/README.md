# 04.10 Check Subtree

## Description

Determine whether one tree is a subtree of another. This maps to LCCI 4.10.

Typical interview angle:
- Walk the larger tree and compare structure/value matches.
- Use a same-tree helper for exact subtree checks.
- Consider null subtree behavior explicitly.

Reference page: https://leetcode.doocs.org/en/lcci/4.10/
