# 04.04 Check Balance

## Description

Check whether a binary tree is height-balanced. This maps to LCCI 4.4.

Typical interview angle:
- Post-order recursion returns subtree height.
- Use sentinel (like -1) for early failure.
- Avoid recomputing heights repeatedly.

Reference page: https://leetcode.doocs.org/en/lcci/4.4/
