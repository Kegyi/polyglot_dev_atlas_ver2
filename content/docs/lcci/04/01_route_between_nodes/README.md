# 04.01 Route Between Nodes

## Description

Given a directed graph, determine whether there is a route from node A to node B. This maps to LCCI 4.1.

Typical interview angle:
- Build adjacency lists.
- Use BFS/DFS with visited tracking.
- Avoid infinite loops in cyclic graphs.

Reference page: https://leetcode.doocs.org/en/lcci/4.1/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
true
false
```
