# 02.06 Palindrome Linked List

## Description

Check whether a singly linked list is a palindrome. This maps to LCCI 2.6.

Typical interview angle:
- Use slow/fast pointers to locate midpoint.
- Reverse second half in-place.
- Compare both halves node by node.

Reference page: https://leetcode.doocs.org/en/lcci/2.6/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
true
true
```
