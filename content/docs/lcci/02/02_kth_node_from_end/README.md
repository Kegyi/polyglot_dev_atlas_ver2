# 02.02 Kth Node From End

## Description

Return the value of the kth node from the end of a singly linked list (k is 1-indexed from the tail). This maps to LCCI 2.2.

Typical interview angle:
- Two-pointer gap technique.
- Single pass after creating a k-step gap.
- Clarify assumptions about valid k.

Reference page: https://leetcode.doocs.org/en/lcci/2.2/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
4
```
