# 02.05 Sum Lists

## Description

Given two numbers represented by linked lists in reverse order, add them and return the sum as a linked list in the same reverse order.

Typical interview angle:
- Simulate elementary addition digit by digit.
- Handle different lengths and trailing carry.
- Keep memory usage linear in output size.

Reference page: https://leetcode.doocs.org/en/lcci/2.5/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
2 1 9
```
