# 16.16 Sub Sort

## Description

Given an integer array, return the smallest subarray `[m, n]` that, if sorted, makes the whole array sorted. If already sorted, return `[-1, -1]`.

- [C++](cpp.cpp)
- [Python](python.py)
- [Go](go.go)
- [TypeScript](typescript.ts)
- [Scala](scala.scala)

## Expected output

```
[3, 9]
```
