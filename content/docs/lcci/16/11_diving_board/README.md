# 16.11 Diving Board

## Description

You have planks of two lengths, `shorter` and `longer`. Return all possible total lengths of a board made of exactly `k` planks.

## Implementations

- [C++](cpp.cpp)
- [Python](python.py)
- [Go](go.go)
- [TypeScript](typescript.ts)
- [Scala](scala.scala)

## Expected output

```
[3, 4, 5, 6]
```
