# 16.22 Langtons Ant

## Description

Simulate Langton's ant for `K` moves on an infinite white grid.
- White cell: flip to black, turn right, move forward
- Black cell: flip to white, turn left, move forward

Return the smallest board containing all black cells and the ant, where the ant is represented by one of `R`, `D`, `L`, `U`.

- [Python](python.py)
- [Go](go.go)
