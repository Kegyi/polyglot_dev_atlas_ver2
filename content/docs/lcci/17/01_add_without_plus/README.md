# 17.01. Add Without Plus

## Description

Write a function that adds two numbers. You should not use `+` or any arithmetic operators.

**Example:**

```
Input: a = 1, b = 2
Output: 3
```

- [C++](cpp.cpp)
- [Python](python.py)
- [Go](go.go)
- [TypeScript](typescript.ts)

```

1
-3
```
