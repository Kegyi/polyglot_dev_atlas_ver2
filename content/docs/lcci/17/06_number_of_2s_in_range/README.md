# 17.06. Number Of 2s In Range

## Description

Write a method to count the number of 2s between 0 and n (inclusive).

**Example:**

```
Input: n = 25
Output: 9
```

**Explanation:** 2, 12, 20, 21, 22, 23, 24, 25 contain digit 2; 22 has two 2s.
