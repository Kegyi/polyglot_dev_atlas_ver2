# 17.05. Find Longest Subarray

## Description

Given an array filled with letters and numbers, find the longest subarray with an equal
number of letters and numbers.

Return the subarray. If there are multiple answers, return the one that occurs first.

**Example 1:**

```
Input: ["A","1","B","C","D","2","3","4","E","5","F","G","6","7","H","I","J","K","L","M"]
Output: ["A","1","B","C","D","2","3","4","E","5","F","G","6","7"]
```

**Example 2:**

```
Input: ["A","A"]
Output: []
```
