# 17.08. Circus Tower

## Description

A circus is designing a tower routine consisting of people standing atop one another's
shoulders. For practical and aesthetic reasons, each person must be both shorter and
lighter than the person below.

Given two arrays, `height` and `weight`, return the maximum number of people in such a tower.

**Example 1:**

```
Input: height = [65,70,56,75,60,68], weight = [100,150,90,190,95,110]
Output: 6
```

**Example 2:**

```
Input: height = [65,70,56,75], weight = [100,150,90,190]
Output: 4
```
