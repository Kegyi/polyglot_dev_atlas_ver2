# 17.11. Find Closest

## Description

Given an array of words and two different words `word1` and `word2`, return the shortest
distance between these two words in the list.

**Example 1:**

```
Input: words = ["I","am","a","student","from","a","university","in","a","city"], word1 = "a", word2 = "student"
Output: 1
```

**Example 2:**

```
Input: words = ["aa","bb"], word1 = "aa", word2 = "bb"
Output: 1
```
