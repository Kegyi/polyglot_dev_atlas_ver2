# 17.10. Find Majority Element

## Description

Given an integer array, return the majority element. A majority element is an element that
appears more than `n / 2` times. If no such element exists, return `-1`.

**Example 1:**

```
Input: [1,2,5,9,5,9,5,5,5]
Output: 5
```

**Example 2:**

```
Input: [3,2]
Output: -1
```
