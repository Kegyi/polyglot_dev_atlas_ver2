# 03.02 Min Stack

## Description

Design a stack supporting push, pop, top, and min in constant time. This maps to LCCI 3.2.

Typical interview angle:
- Keep a second stack of minima.
- Push duplicate minima so pops stay correct.
- Every operation is $O(1)$.

Reference page: https://leetcode.doocs.org/en/lcci/3.2/

## Implementations

- cpp.cpp
- go.go
- python.py
- typescript.ts
- scala.scala

## Input

No input file is required.

## Expected output

```text
2
2
3
```
