# 08.04 Power Set

## Description

Write a method to return all subsets of a set. The solution set must not contain duplicate subsets. Elements in a subset must be in non-descending order. The order of subsets in the result does not matter.

**Example:**
```
Input: nums = [1,2,3]
Output:
[ [3],  [1],  [2],  [1,2,3],  [1,3],  [2,3],  [1,2],  [] ]
```

**Note:** `1 <= nums.length <= 10`
