# 10.03 Search Rotate Array

## Description

Given a sorted array that has been rotated an unknown number of times, find `target`. If there is more than one matching element, return the smallest index.

**Example 1:**
```
Input: arr = [15,16,19,20,25,1,3,4,5,7,10,14], target = 5
Output: 8
```

**Example 2:**
```
Input: arr = [15,16,19,20,25,1,3,4,5,7,10,14], target = 11
Output: -1
```
