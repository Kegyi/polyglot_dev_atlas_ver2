# 10.09 Sorted Matrix Search

## Description

Given an `M x N` matrix in which each row and each column is sorted in ascending order, determine whether `target` exists.

**Example:**
```
Matrix:
[
  [1, 4, 7, 11, 15],
  [2, 5, 8, 12, 19],
  [3, 6, 9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]
Target = 5  -> true
Target = 20 -> false
```
