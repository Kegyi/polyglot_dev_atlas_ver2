03. Memory & Scoping

^

Heap vs Stack

Managing where data lives is crucial for performance and preventing leaks. C++ provides manual control, while JavaScript relies on Garbage Collection.

C++ 
// Manual Heap Allocation
int* p = new int(5);
delete p;
JS 
// Managed by GC
let obj = { val: 5 };
obj = null;

In high-level languages, manual deletion is rare. In systems languages, it is a primary responsibility of the developer.
